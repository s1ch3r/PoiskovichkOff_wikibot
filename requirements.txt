Необходимо виртуальное окружение в папке проекта: python -m venv venv
Также использована сторонняя библиотека aiogram, которую необходимо установить при включенном виртуальном окружении(venv\Scripts\activate)(и так для каждой сторонней библиотеки) с помощью: pip install aiogram
Библиотека BeautifulSoup4 (pip install BeautifulSoup4)
библиотека selenium (pip install selenium)
библиотека wikipedia (pip install wikipedia)
библиотака requests (pip install requests)
